Python学习笔记(一)

发表于2023-07-18|更新于2023-07-19
|阅读量:

一、基础数据类型及词法

map()函数：根据指定的函数对指定序列做映射

print函数

  1. 按格式输出：<模板串> % <值元组>
  1. 逗号连接加空格，加号连接无空格
  2. sep与end参数控制输出格式，sep默认为空格，end默认为换行

注意：input函数输入的默认是字符串

标准库

  1. 包含math, random, copy等模块
  2. 要想使用标准库中的某个模块，必须先使用import语句导入该模块；再对模块中的函数进行调用

Python的内置类型

Python 是一种动态类型化语言，无需事先声明变量类型；由解释器根据其值的表示形式自动匹配其类型

在Python程序中，每个数据都是对象，每个对象包括标识（identity）、类型（type）及“值”(value) 。不同类型有不同的操作方法

布尔类型

表达式可以评值为布尔值

数值类型

int, float, complex（复数型）

可用函数将一个十进制数转换其他进制的字符串 bin(), oct(), hex()

Python词法

运算符

成员测试运算符：in, not in, is, is not

逻辑运算符：and, or, not

除法/与整除//

标识符

用户编程时给Python语言描述的对象所起的名字

赋值语句

可在一条语句中给多个变量赋值，或交换变量值

关键字

是某种程序设计语言中事先定义好的确认符，用来组织语言结构，或者定义内建函数

二、Python的基本语法

程序控制结构

if语句
for语句

利用for语句，可以遍历列表、元组或字典中的每个元素（键）

range函数：

  1. 产生的整数数字包含下限，但不包含上限
  2. range函数经常与for语句结合起来使用，指定循环迭代的范围
while语句

序列类型

序列的通用操作

  1. 索引

    负数索引 s[-1]：序列最末尾元素

  2. 分片

    格式：<列表名>[索引1:索引2:步长]

    副本：<列表名>[:]，对副本的更改不影响原序列

  3. 加

    必须是相同类型的序列才能进行连接操作

  4. 乘

    将原序列重复x次，生成一个新的序列

  5. 成员检测

列表
  1. 值的有序序列，每个值由索引来识别
  2. 用一对方括号包裹多个元素，各个元素通过逗号分隔
  3. 特征：可变长度、异构以及任意嵌套
  4. 列表方法：append, count, extend, index, insert, pop, remove, reverse, sort
warning:
  1. 遍历列表时删除元素
  2. 列表初始化

字符串

字符串方法
  1. find, join, lower, upper, replace, split等等
  2. 注意lower, upper, replace方法返回的是字符串的副本

元组

一个元组由数个逗号分割的值组成

字典

字典是Python中唯一内建的映射类型

特点
  1. 键必须是唯一的
  2. 键必须是任意不可变类型，如数字、字符串或元组，不能是列表
  3. 可以进行字典的嵌套
字典方法

clear, copy, deepcopy, get, keys, pop, values等等

函数

使用def（函数定义）语句创建（或定义）函数。函数的定义一般包括函数名、参数、函数体和返回值等4部分

模块

当程序规模较大时，将不同功能的程序段存储于不同的文件中，每个文件称为一个模块。在其他程序中导入该模块，即可完成相应功能

文件

文件句柄

Python通过文件句柄（file handle）访问磁盘文件，实现操作系统无关性

open函数的使用

格式：open(name[,mode[,buffering]])

注意区分’w’只写模式和’a’追加模式

读取文件
  1. fileHandle.read([size=-1])，从文件读取size个字符；当未给定size或size为负值时，读取文件剩余的所有字符，并将文件内容以字符串返回
  2. fileHandle.readline([size])，从文件中读取并返回下一行（包括行结束符），返回字符串；若size有定义则返回size个字符
  3. fileHandle.readlines([size])，读取一个文件中的所有行，返回列表类型，其中每个元素是文件的一行；若size有定义则返回size个字符
写入文件
  1. fileHandle.write(s)，将字符串s写入文件，返回写入字符的长度
  2. fileHandle.writelines(SL)，SL为字符串的序列，将该序列的每个元素一次性写入文件
  3. fileHandle.close()，关闭文件
