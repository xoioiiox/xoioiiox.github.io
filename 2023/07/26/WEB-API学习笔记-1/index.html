建立项目

Web API是网站的一部分，用于与使用具体URL请求特定信息的程序交互。这种请求称为API调用。

Django简介

MVC模型：Model（模型）+ View（视图）+ Controller（控制器）设计模式

在Django中创建项目

让Django新建一个名为mysite的项目：django-admin startproject mysite

创建数据库

Django将大部分与项目相关的信息存储在数据库中，因此需要创建一个 供Django使用的数据库：python manage.py migrate

  • 首次执行命令migrate 时，将让Django确保数据库与项目的当前状态匹配。

  • 在使用SQLite（后 面将详细介绍）的新项目中首次执行这个命令时，Django将新建一个数 据库。

查看项目

python manage.py runserver

Django启动一个名为development server的服务器，使得能够查看系统中的项目，指出项目的URL(Uniform Resource Locator)。

Django项目由一系列应用程序组成，它们协同工作让项目成为一个整体。

python manage.py startapp app_name：让Django搭建创建应用程序所需的基础设施
